> Data structures provide **a means** to manage large amounts of data **efficiently** for uses such as large databases and internet indexing services. Usually, efficient data structures are key to designing efficient algorithms. [[wiki](https://www.wikiwand.com/en/Data_structure)]

## Why 

**Data structures** serve as the basis for abstract data types (ADT). The ADT defines the **logical** form of the data type. The data structure implements the **physical** form of the data type.

Different types of data structures are suited to different kinds of applications, and some are highly specialized to specific tasks. For example, relational databases commonly use B-tree indexes for data retrieval, while compiler implementations usually use hash tables to look up identifiers.

## What

* **Data Structure**: a **collection** of data values, the **relationships** among them, and the functions or **operations** that can be applied to the data.
* **Data Type**: a data type or simply type is an attribute of data which tells the compiler or interpreter how the programmer intends to use the data. Most programming languages support basic data types of integer numbers, floating-point numbers, characters and Booleans.
